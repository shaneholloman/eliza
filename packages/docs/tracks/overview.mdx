## How the dimensions compose

These aren't silos. A typical project crosses several:

- A **Plugin** is consumed by an **Agent** (Runtime), wrapped in an **App**, optionally deployed to **Cloud**.
- A **Robot** policy loads an Eliza-1 model from the **Runtime**.
- The **Chip** runs the **OS**, which hosts the **Runtime**, which runs the **App**.

Start with the dimension closest to your output artifact and follow the links inward.
