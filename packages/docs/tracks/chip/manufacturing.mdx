The hardware bring-up artifacts live alongside the SoC source:

## Board

`upstreams/research/chip/board/` — KiCad projects for:

- The phone reference design.
- A PMIC daughtercard for power management.
- A demo board for first-silicon validation.
- The FPGA mapping (VCU118) used pre-silicon.

## Package

`upstreams/research/chip/package/` —

- Pinout — mapping from die pads to package balls.
- Bonding — wire / flip-chip bonding plan.
- Wi-Fi interface manifests for production builds.

## Thermal & signoff

Thermal analysis and final package signoff live under `pd/` and `package/`. The signoff manifests pin tool + version + acceptance criteria the same way the digital signoff does.
